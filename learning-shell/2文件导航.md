# 文件导航

- **pwd**：输出当前文件夹
- **cd**：更改当前文件夹
- **ls**：列出文件夹内容

## 当前文件夹

**输出当前文件夹：**

```bash
me@ubuntu16.04:~$ pwd
...
```

## 列出文件

**列出当前文件夹下的文件和文件夹：**

```bash
me@ubuntu16.04:~$ ls
...
```

## 跳转

**跳转到制定文件夹：**

```bash
me@ubuntu16.04:~$ cd /usr/bin
me@ubuntu16.04:/usr/bin$ cd pwd
```

**上例使用的是绝对路径，也可使用相对路径：**

```bash
me@ubuntu16.04:~$ cd /usr/bin
me@ubuntu16.04:/usr/bin$ cd /usr
```

等价于：

```bash
me@ubuntu16.04:~$ cd /usr/bin
me@ubuntu16.04:/usr/bin$ cd ..
```

`..`表示父文件夹，`.`表示当前文件夹。

**cd** 命令的其它用法

命令        | 效果
:-------- | :--------------------
`cd`      | 跳转到home文件夹
`cd -`    | 跳转到上次打开的文件夹
`cd ~用户名` | 多用户情况下，跳转到该用户的home文件夹
