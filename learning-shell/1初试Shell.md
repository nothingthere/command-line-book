# 初试Shell

命令行打开快捷键： <kbd>Ctrl+Alt+T</kbd>

```bash
me@ubuntu16.04:~$
```

以上格式含义为：用户名@电脑名:当前文件夹和“$”。

如果不是"$"，而是以“#”结尾表示为超级用户权限。

导航：

-   <kbd>Up</kbd>：上一个命令
-   <kbd>Down</kbd>：下一个命令
-   <kbd>Left</kbd> <kbd>Right</kbd>：光标左右移动

## 一些简单命令

**获取当前时间：**

```bash
me@ubuntu16.04:~$ date
...
```

**当前日期：**

```bash
me@ubuntu16.04:~$ cal
...
```

**硬盘剩余空间：**

```bash
me@ubuntu16.04:~$ df
...
```

**内存剩余空间：**

```bash
me@ubuntu16.04:~$ free
...
```

**关闭命令行：**

```bash
me@ubuntu16.04:~$ exit
```
